# Firefox Monitor SSR/Client Recreation

## Summary
This is a test recreation of [Firefox Monitor](https://github.com/mozilla/blurts-server) with Server-side rendering built using [Next.js](https://nextjs.org/)

The test server is hosted on heroku at 

## Development

### Requirements

* [Node](https://nodejs.org) 14.15.1 LTS (recommended, w/ npm)

### Install

1. Clone and change to the directory:

    ```bash
   git clone https://github.com/stanleykerr/next-moz-test
   cd next-moz-test
   ```
   
2. Install dependencies:

    ```bash
   yarn install
   # or
   npm install
    ```

### Run

1. Run the server:

    ```bash
    npm run dev
    # or
    yarn dev
    ```

2. Open [http://localhost:3000/](http://localhost:3000/) with your browser to see the current progress.
   
   A more complete recreation can be seen on the about page ([http://localhost:3000/about](http://localhost:3000/about))
